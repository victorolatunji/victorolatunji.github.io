# Canada Provincial Income Tax Calculator (Blazor WebAssembly)

## Overview
A Blazor WebAssembly application that calculates federal and provincial income tax
for all ten Canadian provinces using the 2025 marginal tax brackets.

This project was created as part of the COSC 2100 â€“ Object Oriented Programming II
.NET Topic Assignment.

## Features
- Built entirely with **Blazor WebAssembly**
- Calculates:
  - Federal income tax
  - Provincial income tax
  - Total tax
  - Effective tax rate
- Uses official 2025 tax brackets from CRA and provinces
- Runs fully in the browser

## Technologies Used
- .NET 8
- Blazor WebAssembly
- C#

## How to Run
1. Clone the repository
2. Open the solution in Visual Studio 2022
3. Set the Blazor project as Startup Project
4. Run (Ctrl + F5)
